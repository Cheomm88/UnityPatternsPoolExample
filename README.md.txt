1- Descargar el paquete (Window -> Package Manager -> Dentro de Unity Registry -> TimeLine)
2- Crear un GameObject vacío, lo llamamos "TimeLine" y seleccionarlo.
3- Window -> Sequencing -> TimeLine en esta ventana le damos a Create para crear un TimeLine Asset.
	Un TimeLine Asset almacena la información sobre la cinemática.
4- En la ventana de TimeLine encontramos un "+" que nos permite crear diferentes tipos de pistas.
